O que é HTML?

Conceito

É uma linguagem de marcação utilizada na construção de páginas na Web

Estrutura básica de um documento

Cabeçalho

Dentro do cabeçalho podemos encontrar os seguintes elementos: title, style, scripts, links.

Corpo

Dentro do corpo podemos encontrar outros vários marcadores que irão moldar a página como por exemplo: h1,h2,h3..., p, br, table.
